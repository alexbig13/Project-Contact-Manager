# Book & Movie Tracker

## Description

This project is a Book & Movie Tracker built with Vanilla JavaScript.
It allows users to manage their personal collection of books and movies, add reviews and ratings, search, filter, and sort items while keeping the application state synchronized with the UI.

## Features

- Add books and movies
- Delete items
- Mark books as read or movies as watched
- Search by name
- Sort by year, category, or rating
- Filter by books or movies
- Add reviews and ratings
- Display detailed information for the selected item
- Dynamic DOM rendering
- State management using JavaScript objects and arrays
